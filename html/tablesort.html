Make sortable HTML tables with tableSort.js!

(1) Grab a copy of tableSort here: 

pastebin.com/raw/FRY5pYNt

(2) Source the javascript in the head of your HTML document: 

<script src="tableSort.js"></script>

(3) Just add the following class to a <table> header: 

class="sortable"

You can see this effect in action in the table below (click the column headers to demo the sorting feature).
